# v2 — The researchable question (2026-06-14)

Refined with the constraints that actually mattered, which launched a deep, multi-source research sweep:

```
Probably use JS or something. I want to keep costs down so free should cover
as much as possible. Which platforms allow you to run commercial projects for
free? I know Vercel requires Pro. Which is genuinely best? Tell me what big
companies started on before expanding — and focus on the best genuinely-free
options for someone in my position right now.
```

**What changed:** pinned the stack (JS / Next.js), set the real priority (free covers as much as possible), and reframed the big-company angle from "what they run now" to what they *started* on — which is the part that's genuinely instructive (almost everyone began on cheap/free infra).

**What it produced:** a 250+ source comparison ranking platforms by whether their free tier permits commercial use — ruling out Vercel Hobby and GitHub Pages — and a single decision: **Cloudflare**, with the OpenNext adapter and 3 MiB free Worker cap noted as the things to watch.
